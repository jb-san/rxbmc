rXBMC
=====
A gem to controll an xbmc server.

## Installation

Just run gem install rxbmc

    gem 'rxbmc'



## Usage
first time running rxmbc you will be asked to fill in connection details to the xbmc server.


example
    rxmbc volume 43
  setts the volume to
